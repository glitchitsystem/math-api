üîê Authentication

POST /auth/login

Authenticate and receive a JWT token

Request:

POST /auth/login Content-Type: application/json { "username": "admin", "password": "password" }

Success Response (200):

{ "message": "Login successful", "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...", "expiresIn": "24h" }

üî¢ Math Endpoints

GET /math/calculate

Basic operations on two numbers

üîí Authentication Required: Include Authorization: Bearer <token> header

Parameters:

  • operation: add, subtract, multiply, divide
  • a: First number
  • b: Second number

Example:

GET /math/calculate?operation=add&a=10&b=5 Authorization: Bearer YOUR_TOKEN_HERE

Response:

{ "operation": "add", "operands": { "a": 10, "b": 5 }, "result": 15, "timestamp": "2025-11-02T12:00:00.000Z" }

POST /math/calculate

Operations on arrays of numbers

üîí Authentication Required

Supported Operations:

  • sum: Add all numbers
  • product: Multiply all numbers
  • average: Calculate mean
  • max: Find maximum
  • min: Find minimum

Request:

POST /math/calculate Authorization: Bearer YOUR_TOKEN_HERE Content-Type: application/json { "operation": "sum", "numbers": [1, 2, 3, 4, 5] }

Response:

{ "operation": "sum", "numbers": [1, 2, 3, 4, 5], "result": 15, "count": 5, "timestamp": "2025-11-02T12:00:00.000Z" }

PUT /math/power

Calculate power operations (base^exponent)

üîí Authentication Required

Request:

PUT /math/power Authorization: Bearer YOUR_TOKEN_HERE Content-Type: application/json { "base": 2, "exponent": 8 }

Response:

{ "operation": "power", "base": 2, "exponent": 8, "result": 256, "expression": "2^8", "timestamp": "2025-11-02T12:00:00.000Z" }

GET /math/factorial

Calculate factorial of a number (n!)

üîí Authentication Required

Parameters:

  • n: Non-negative integer (0-170)

Example:

GET /math/factorial?n=5 Authorization: Bearer YOUR_TOKEN_HERE

Response:

{ "operation": "factorial", "input": 5, "result": 120, "expression": "5!", "timestamp": "2025-11-02T12:00:00.000Z" }

Test power operation curl -X PUT http://localhost:3000/math/power \ -H "Authorization: Bearer $TOKEN" \ -H "Content-Type: application/json" \ -d '{"base": 3, "exponent": 4}'

üìÆ Postman Collection

Import the ready-made Postman collection for easy testing:

Files Available:

  • Math_API_Collection.postman_collection.json - Complete Postman collection
  • POSTMAN_TROUBLESHOOTING.md - Troubleshooting guide for common issues
  • test-api.sh - Shell script for automated testing

How to Import in Postman:

  1. Open Postman
  2. Click "Import" button
  3. Select "Math_API_Collection.postman_collection.json"
  4. Run "Login" request first to get token
  5. Token is automatically saved for other requests
